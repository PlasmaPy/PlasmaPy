Revised the `~plasmapy.diagnostics.langmuir.Characteristic.get_unique_bias` function to increase speed. Instead of iterating over all bias values and averaging all currents with that bias, the function passes if there are no duplicate biases; if there are duplicate biases, averaging is only performed for those duplicate bias values. Speeds up `~plasmapy.diagnostics.langmuir.Characteristic` math such as subtraction.
